
# Trackme

This project is an application designed to track personal food intake and body weight, aimed at helping users effectively monitor and manage their weight.

# Project Objective
* For personal use  
* To study system design and development with a focus on security and scalability  
* To learn and enhance knowledge of **Spring Security and JWT**  
* To learn and improve SQL skills using PostgreSQL  
* To study Docker usage, server management with Linux, and deployment on AWS EC2
